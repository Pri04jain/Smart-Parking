# ParkEasy - Smart Parking System

## 🚀 Features
- Real-time parking detection using YOLOv8
- Slot-wise occupancy detection
- Live availability tracking

## 🛠 Tech Stack
- Python, OpenCV
- YOLOv8
- PKLot Dataset

## ▶️ How to Run
1. pip install -r requirements.txt
2. python main.py
